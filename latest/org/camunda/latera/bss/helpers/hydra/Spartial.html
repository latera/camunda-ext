        • void calcTargetingData(Map input = [:])

          Get equipment satellite targeting data and fill up execution variables

          Input execution variables:

          • homsOrderData*Equipment*Id BigInteger
          • homsOrderData*Equipment**%BindAddrType%Address String. Address is used to determine targeting data. Used only if address argument is empty
          • homsOrderData*Equipment*AntennaDiameter String

          Output execution variables:

          • homsOrderData*Equipment*Azimuth String
          • homsOrderData*Equipment*AntennaPhi String
          • homsOrderData*Equipment*DirectPolarization String
          • homsOrderData*Equipment*DirectRFCC String
          • homsOrderData*Equipment*DirectdB String
          • homsOrderData*Equipment*DirectRayNumber String
          • homsOrderData*Equipment*DirectEIRP String
          • homsOrderData*Equipment*ReversedB String
          • homsOrderData*Equipment*ReversedBK String
          • homsOrderData*Equipment*ReverseGT String
          Parameters:
          address - String. Address is used to determine targeting data. Optional. Default: empty string
          bindAddrType - String. Equipment-address bind type without 'BIND_ADDR_TYPE_' part. Optional. Default: 'Serv'
          spartialService - HTTPRestProcessor. Instance of spartial REST service which is used to get targeting data. Optional. Default: null
          spartialServiceUrl - String. URL of spartial REST service which is used if spartialService == null
          prefix - String. Targeting data variables prefix. Optional. Default: empty string
          equipmentPrefix - String. Equipment prefix. Optional. Default: empty string
          equipmentSuffix - String. Equipment suffix. Optional. Default: empty string
