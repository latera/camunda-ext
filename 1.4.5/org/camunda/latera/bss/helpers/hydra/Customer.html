          Add application subscription to a customer and fill up execution variables

          • homsOrderData*Customer%Application%Password BigInteger

        • Boolean deleteCustomerAddParam(Map input = [:])

          Delete base subject additional parameter value and fill up execution variables

          Input execution variables:

          • homsOrderData*CustomerId BigInteger
          • homsOrderData*Customer*%Param%Id BigInteger. if additional parameter value is ref id
          • homsOrderData*Customer*%Param% Any type, if additional parameter value is not a ref or ref code is used instead

          Output execution variables:

          • homsOrderData*Customer*%Param%Deleted Boolean. Same as return value
          Parameters:
          param - String. Additional parameter short code (=variable part name) ('Param' for 'SUBJ_VAL_Param')
          code - String. Additional parameter full code (if it does not start from 'SUBJ_VAL_')
          customerPrefix - String. Customer prefix. Optional. Default: empty string
          prefix - String. Additional parameter prefix. Optional. Default: empty string
          force - Boolean. For multiple additional parameters. If you need to remove only a value which is equal to one stored in the input execution variable, pass false. Otherwise method will remove additional param value without check
          Returns:
          True if additional parameter value was deleted successfully, false otherwise
