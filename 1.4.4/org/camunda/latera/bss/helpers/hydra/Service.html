        • void fetchServiceSubscription(Map input = [:])

          Get subscription data by its customer id, account id, contract id, service id, equipment id and dates, and fill up execution variables

          Input execution variables:

          • homsOrderData*CustomerId BigInteger
          • homsOrderData*AccountId BigInteger
          • homsOrderData*ContractId BigInteger. Optional, used only if contractType == 'contract'
          • homsOrderData*ContractAppId BigInteger. Optional, used only if contractType == 'contractApp'
          • homsOrderData*AddAgreementId BigInteger. Optional, used only if contractType == 'addAgreement'
          • homsOrderData*EquipmentId BigInteger
          • homsOrderData*ServiceId BigInteger

          Output execution variables:

          • homsOrderData*Equipment*Service*SubscriptionId BigInteger
          • homsOrderData*Equipment*Service*ParSubscriptionId BigInteger
          • homsOrderData*Equipment*Service*PrevSubscriptionId BigInteger
          Parameters:
          prefix - String. Subscription prefix. Optional. Default: empty string
          customerPrefix - String. Customer prefix. Optional. Default: empty string
          accountPrefix - String. Account prefix. Optional. Default: empty string
          contractPrefix - String. Contract, contract app or add agreement prefix. Optional. Default: empty string
          contractType - String 'contract', 'contractApp' or 'addAgreement'. Optional. Default: 'contract'
          servicePrefix - String. Service and price line prefix. Optional. Default: empty string
          equipmentPrefix - String. Equipment prefix. Optional. Default: empty string
          parSubscriptionPrefix - String. Parent subscription prefix. Optional. Default: empty string
          prevSubscriptionPrefix - String. Previous subscription prefix. Optional. Default: empty string
          beginDate - Any date type. Subscription begin date. Optional. Default: null
          endDate - Any date type. Subscription end date. Optional. Default: null
          operationDate - Any date type. Date which should overlap subscription begin and end dates. Optional. Default: current datetime
          isClosed - Boolean. If false search only non-closed subscriptions, if true search only closed subscriptions, if null - disable filter. Optional. Default: false
          onlyParent - Boolean. If true search only parent subscriptions, if false disable filter. Optional. Default: true
